## 📋 Текстовая версия для слайда

### Основные компоненты:

1. **Пользователь** → загружает файл или вводит URL

2. **Frontend (React)** → интерфейс загрузки и отображения

3. **Backend (NestJS)** → обработка запросов

4. **Video Service** → извлечение аудио и кадров

5. **Два параллельных потока:**
   - **Аудио путь:** Yandex STT → Транскрипция
   - **Визуальный путь:** Yandex Vision → Описание кадров → Yandex GPT (улучшение)

6. **Yandex GPT Service** → объединение данных → "Паспорт стиля"

7. **Генерация сценария:** Тема + Паспорт стиля → Yandex GPT → Готовый сценарий

## 🎨 Визуальные элементы для слайда

### Цветовая схема:
- **Пользователь** — синий
- **Frontend** — зеленый
- **Backend** — оранжевый
- **AI сервисы** — фиолетовый
- **Результаты** — желтый

### Стрелки:
- **Толстые стрелки** — основные потоки данных
- **Тонкие стрелки** — вспомогательные операции
- **Пунктирные стрелки** — опциональные пути
